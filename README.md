# hiera-pfunc hiera backend

This is a hiera backend to call puppet functions from hiera.

## Usage
Do not use it if you do not know what you are doing.

## Hints about usage
* needs an entry in hiera.yaml (example see in this modules hiera.yaml)
* for how to use, see common.yaml in data directory for examples
* for a demo use ```include pfunc```

## Warning
Use of this backend is neither tested nor do I recommend using it.
